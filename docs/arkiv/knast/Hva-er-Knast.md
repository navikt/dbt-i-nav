# Hva er Knast?

!!! warning "Arkiv"
    Denne siden er arkivert og er ikke del av anbefalt oppsett lenger. Innholdet beholdes kun for historikk og overgangsbehov. For nytt arbeid, bruk de aktive sidene for Knast og DVH.

Knast er ditt eget personlig utviklingsmiljø for analyse og behandling av data som bygges på toppen av [Google Cloud Workstations](https://cloud.google.com/workstations) i Nav sin Google Cloud Platform. Dokumentasjon om Knast finnes på [nada sine sider](https://docs.knada.io/analyse/knast/generelt/).
