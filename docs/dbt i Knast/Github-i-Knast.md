# Github i Knast
Git og gh (github CLI) kommer preinstallert i Knast.
Det anbefales å bruke `gh` for å jobbe mot Github.

- **Authentiser mot Github:** Kjør `gh auth login`. Da Github credentials blir lagret trygt i os keyring på Knast.

Nå bruker kan jobbe på Github via både `git`, `gh` eller innebygd Source Control i code-oss/VS-Code.
